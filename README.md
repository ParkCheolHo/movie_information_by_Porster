# Movie Information

    포스터로 영화를 판단하는 형식의 영화 커뮤니티 홈페이지 입니다.

### 개발환경
    * node.js 4.4.2
    * mysql.5.7.12

### 현재 사용 가능한 기능
    * 회원가입
    * 로그인()
    * 글쓰기(수정, 삭제 가능)
    * 영화 상세정보 보기

### 현재 만들고 있는 기능
    * 내가 본 영화 트래킹
    * 내가 쓴 글 트래킹    
    * 아이디, 비밀번호 찾기
    * 게시판 댓글 기능
